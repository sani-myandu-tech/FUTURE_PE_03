FUTURE_PE_03 – AI SEO CONTENT CLUSTER GENERATOR

PROJECT OVERVIEW
A complete AI-powered SEO content strategy for Urban Bites Café,
Durban — built using structured prompt engineering to drive local
search traffic, website authority, and real business growth.

BUSINESS SELECTED
  Business      : Urban Bites Café
  Location      : Durban, South Africa
  Target Market : Students and young professionals

OBJECTIVE
To build a high-performing SEO content system including:
  * 1 pillar blog targeting a high-traffic primary keyword
  * 4 supporting blogs forming a complete content cluster
  * SEO-optimised prompts for consistent AI content generation
  * Local SEO targeting strategy for Durban search results

KEYWORD STRATEGY
  Primary Keyword:
    * Affordable café in Durban

  Secondary Keywords:
    * Cheap eats Durban
    * Budget-friendly restaurants Durban
    * Best cafés in Durban

  Search Intent:
    * Local + Transactional + Informational

CONTENT CLUSTER STRATEGY
The pillar blog targets a broad keyword and acts as the authority
page. Supporting blogs capture long-tail searches and link back to
the pillar, building internal link equity and topical relevance.

  [PILLAR BLOG]
  The Best Affordable Cafés in Durban — Ultimate Guide

  [SUPPORTING BLOGS]
  01. Where to Eat in Durban on a Student Budget
  02. Top Budget-Friendly Restaurants Near Durban Universities
  03. Cheap Eats Durban: Hidden Gems You Need to Try
  04. Why Urban Bites Café is Durban's Best Kept Secret

  Benefits of this structure:
    * Improved Google ranking
    * Higher website authority
    * Stronger user engagement

TOOLS USED
  * ChatGPT — prompt engineering
  * Google Search — keyword inspiration

KEY SKILLS DEMONSTRATED
  * SEO strategy development
  * Prompt engineering for content generation
  * Local SEO optimisation
  * Conversion-focused writing

REAL-WORLD APPLICATION
This system can be used by:
  * Local businesses
  * Digital marketing agencies
  * Freelancers offering SEO services

CONCLUSION
  This project reflects real-world SEO practices applied through
  structured prompt engineering. The content cluster model —
  combining a high-authority pillar blog with targeted supporting
  articles — is an industry-standard strategy used to generate
  sustainable organic traffic, qualified leads, and measurable
  business growth.
